## How It Works

1. **Click a button** — opens Rebyte with the skill and prompt pre-loaded
2. **Sign in** — free account, no credit card required to start
3. **Watch it run** — the agent executes in a cloud VM with full capabilities
4. **Get results** — files, deployed apps, reports, and more are delivered to you

## Customize Any Prompt

Every button above is just a URL. Change the prompt to whatever you want:

```
https://app.rebyte.ai/new?prompt=Use%20the%20SKILL_NAME%20skill.%20YOUR%20PROMPT%20HERE
```

## Add a "Run on Rebyte" Button to Your Repo

Want to let users run your skill with one click? Add this to your README:

```markdown
[![Run on Rebyte](https://img.shields.io/badge/▶_Run_on_Rebyte-D4A654?style=for-the-badge)](https://app.rebyte.ai/new?prompt=Use%20the%20YOUR_SKILL%20skill%20from%20github.com/YOUR_ORG/YOUR_REPO.%20YOUR%20DEFAULT%20PROMPT.)
```

Or embed the button on any website:

```html
<script src="https://rebyte.ai/embed.js"
  data-prompt="Use the YOUR_SKILL skill from github.com/YOUR_ORG/YOUR_REPO. YOUR DEFAULT PROMPT."
  data-text="Run on Rebyte"
  data-theme="light"
  data-size="md">
</script>
```
